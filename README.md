## General info
"A basic movie database interacting with external API (http://www.omdbapi.com/)."

## Technologies
* Django
* Django REST framework
* Docker
* Nginx
* Redis
* Celery
	
## Setup
```
$ docker-compose build
$ docker-compose up 
```

The app is available at: 

```
http://localhost:8000/api/v1
```
